```bash
# Clone este repositório
$ git clone <https://github.com/Robson-Carvalho/Clone-Spotiy.git>

# Acesse a pasta do projeto no terminal/cmd
$ cd Clone-Spotify

# Vá para a pasta raíz 
$ cd start index.html
```
